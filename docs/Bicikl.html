Бицикл са сигнализацијом¶


  • Са колико година дете сме самостално да вози бицикл?

  • Шта је неопходно да сваки бицикл има од сигнализације?

  • Којих све правила треба да се придржавамо ако желимо да наше учешће у саобраћају буде безбедно?

  • Могу ли савремене технологије учинити нашу вожњу безбеднијом?

Ако сте пронашли одговоре на претходна питања и проучили Закон о безбедности саобраћаја на путевима, можемо да наставимо са радом! ☺️

Употребићемо микробит уређаје како бисмо бицикл опремили додатним сигналним уређајима и на тај начин учинили вожњу безбеднијом.


Шта нам је потребно?

  • три микробита

  • траке или материјал за футролу, како бисмо поставили уређаје на бицикл

from microbit import *                                     
import radio

radio.on()

while True:
    # mikrobit se postavlja na volan 'licem' od nas,
    # tako da nam je dugme A na desnoj, a B na levoj strani
    if button_b.was_pressed()
        radio.send("L")
        # gledano spreda, na levo pokazuje strelica ARROW_E
        display.show(Image.ARROW_E)
    elif button_a.was_pressed()
        radio.send("D")
        # gledano spreda, na desno pokazuje strelica ARROW_W
        display.show(Image.ARROW_W)
    else:
        display.clear()
    sleep(1000)

from microbit import *                                     
import radio

radio.on()

while True:
    skreni = radio.receive()
    if skreni is not None:
        if skreni == "L":
            for i in range(4):
                display.show(Image.ARROW_W)
                sleep(1000)
                display.clear()
                sleep(500)

Размислите како бисте могли да примените технологију да учините вожњу бициклом још безбеднијом. У примеру који смо приказали, микробит уређај поставили смо на кацигу и сигнализацију укључивали благим нагињањем главе у леву или десну страну. Развијте систем који ће омогућити да се сигнализација активира закретањем кормана бицикла. Додајте могућност да вас микробит упозори да укључите светла ако је смањена видљивост.

Свакако, током вожње се, без обзира на додатне сигналне уређаје, морате обавезно придржавати правила прописаних законом! Уопште, приликом израде било којег техничког уређаја морате водити рачуна да његова примена буде у складу са прописима везаним за област у којој се примењује.
