Једно од програмских решења може бити следеће::
	
	from microbit import *

	broj_koraka = 0

	while True:
		if accelerometer.was_gesture("shake"):
			broj_koraka += 1
	
		if button_a.was_pressed():
			if broj_koraka > 3000:
				display.show(Image.YES)
			else:
				for x in range(3):
					display.show(Image.NO)
					sleep(500)
					display.clear()
					sleep(500)
		if button_b.was_pressed():
			display.scroll(broj_koraka)
		if button_a.is_pressed() and button_b.is_pressed():
			broj_koraka = 0

Програм за коцкицу коју "бацамо"::

	from microbit import *
	import radio

	radio.on()  # укључујемо радио
	# radio.config(channel=19)
	# radio.config(power=7)
	
	while True:
		gesture = accelerometer.current_gesture()
		if gesture == "shake":
			radio.send("B")

|

Програм за преостале коцкице::

	from microbit import *
	import radio
	import random

	radio.on()  # укључујемо радио
	# radio.config(channel=19)
	# radio.config(power=7)
	while True:
		incoming = radio.receive()
		if incoming is not None:
			if incoming == "B":
				broj = random.randint(1, 6)
				display.show(str(broj))

Микробит који прикупља податке и приказује број возила::

	from microbit import *
	import radio

	radio.on()  
	# radio.config(channel=19)
	# radio.config(power=7)
	br_vozila = 0
	while True:
		display.show(str(br_vozila))
		incoming = radio.receive()
		if incoming is not None:
			if incoming == "U":
				br_vozila += 1
			elif incoming == "I":
				br_vozila -= 1
		sleep(100)
